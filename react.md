# [React](https://facebook.github.io/react/)

> JUST THE UI

MVCフレームワークとは違って、MVCのVの部分、UIコンポーネントをつくるライブラリ。

> VIRTUAL DOM

DOMへの変更を最小限にするため、virtual domをJavaScriptオブジェクトで持ち、レンダリングするときに変更部分のみを実際のDOMに反映する。

> DATA FLOW

データの流れを１方向にするようにして、プログラム構造をわかりやすくする。
