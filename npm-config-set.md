# npm config set

最近、npmはデフォルトで`npm init`のバージョンが`1.0.0`に、ライセンスが`ISC`。

バージョンもライセンスも`npm config set`で変更できる。
```sh
$ npm config set init.version 0.0.1
$ npm config set init.license MIT
```
