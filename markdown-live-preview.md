# Markdwon live preview

## npmで

https://github.com/shime/livedown

### インストール

```
$ npm i -g livedown
```

### ブラウザで開く

```
$ livedown start target.md --open
```


### vimで

https://github.com/shime/vim-livedown

```
NeoBundle 'https://github.com/shime/vim-livedown.git'
```

#### ブザウザで開く

```
:LivedownPreview
```
