# gibo

`.gitignore`を生成するツール。


## Install

```sh
$ brew install gibo
```


## Usage

`Swift` プロジェクトの`.gitignore`を生成する場合、

```sh
$ gibo Swift
```


指定可能な言語等は
```
$ gibo --list
```
で出せる。
